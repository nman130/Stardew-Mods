Menus Everywhere - 0.1.0

This mod allows players to access various menus from anywhere in the game.

Requirements:
Stardew Valley Version 1.4
SMAPI Version 3.5.0

Accessible Menus & Default Keybinds:
Calendar - F1
Wanted Board - F2
Monster Eradication Board - F3

Accessing bounties from outside the community center can be turned off in the config.
